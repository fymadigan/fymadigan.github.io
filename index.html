About Hoopfest

×
Bainbridge Hoopfest was the first major webiste I coded that was designed for the general public and not for a class project. The webiste can be accessed on my GitHub repo or through. About Predictive Perception

×
This was a fun quick side project comissioned by a friend to help explain a philisophical concept of free will. Very simple static HTML page but utilized Javascript, a language I'm not very familiar with to be the main point of this project. The user must click the space bar after they see the green flash as fast as they can while the toggle is on and the same once it's off. The toggle changes the delay in animation and supposedly once a user gets used to the delayed one, they feel a sensation of a lack of free will when it happens so quickly.

About BeatBallot

×
NW Hacks 2019 was my first hackathon and for it my team created BeatBallot. I worked on the front-end using React, HTML, and CSS, effectively turning the sketches into the reality. Within 24 ours, our team developed a fully functional application.
The concept behind BeatBallot is users can host a room that anyone can join via a QR code - oor six alphanumeric string. Once in the room, users can suggest songs to the host playlist that are then voted on by other guests in the room. Once a song recieves more downvotes than upvotes, the song will be axed fromm the playlist. The songs within the playlist will play based on most likes first but if they are tied, they will be played oldest added to newest.

About Access for All

×
This project was for DubHacks 2019. I worked on the front-end using HTML, CSS, and Javascript and turned the Figma low fidelity sketches into a dynamic web app. Our project was announced as top 3 for the Microsoft Sponsor prize.
We wanted our project to help a community and one thing we decided was that websites are often cluttered, confusing, and overstimulating for many but especially for those who identify as autisitic. Our webapp can take any website and then display it in one clean format which encourages consistency and ultimately provides a cleaner viewing experience.
