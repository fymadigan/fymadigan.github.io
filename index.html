About Hoopfest

×
Bainbridge Hoopfest was the first major webiste I coded that was designed for the general public and not for a class project. This event was to take place in March of 2018 as a 3 vs 3 basketball tournament for Kitsap County. This website was to be the go-to place for any information for it and was a project I was thrilled to work on. Unfortunately, the support for it got pulled by administration late in the game, after the webiste was near completion. I include this website on my portfolio because I consider it one of my greatest successes and failures due to the work and dedication I put into trying to make it happen. The webiste can be accessed on my GitHub repo or through here . Please note some functionalities might be broken now - such as the countdown clock as it was designed to go to the date and not optimized to go later.

About Predictive Perception

×
This was a fun quick side project comissioned by a friend to help explain a philisophical concept of free will. Very simple static HTML page but utilized Javascript, a language I'm not very familiar with to be the main point of this project. The user must click the space bar after they see the green flash as fast as they can while the toggle is on and the same once it's off. The toggle changes the delay in animation and supposedly once a user gets used to the delayed one, they feel a sensation of a lack of free will when it happens so quickly.

About BC Jeopardy

×
My senior year end of year project for this course was to create a Jeopardy game to resemble the real one as much as possible. I especially enjoyed this project for its scoreboard that stays updated. This site did experience some bugs especially since the popups were "< a >" link tags so one would have to reset their history in order to "clear the board" but I had a lot of fun putting it together and learning about the LaTex math script.
